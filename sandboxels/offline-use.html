< Sandboxels Offline Use

Sandboxels can be installed for offline use on mobile. See our Mobile Use Guide.

On PC, Sandboxels can only be installed offline on Google Chrome using the steps below.

  1. Open Google Chrome. Other browsers will not work.
  2. Go to the official Sandboxels website in Chrome.
  3. There should be a button underneath the game that says Install Offline. Press it and follow the instructions provided by the browser. You may also see an install option in the address bar.
  4. After installing, Sandboxels can act like a standalone app on your computer without a Wi-Fi connection.
